# ML_from_scratch

## Description
Basic Machine Learning algorithms are implemented including:
  - Linear Regression
  - Logistic Regression
  - Naive Bayes
  - Decision Trees
  - Random Forest
  - k-Nearest Neighbors (kNN)
  - Support Vector Machine (SVM)
  - Principal Component Analysis (PCA)
  - Preceptron
